# Folder Structure

```text
UniversalPromptStudio/
  Backend/
    core/
    domain/
    application/
    infrastructure/
    interfaces/
    implementations/
    repositories/
    services/
    plugins/
    utilities/
  Frontend/
  Database/
  Assets/
  Templates/
  Categories/
  Themes/
  Plugins/
  Docs/
  Tests/
  Scripts/
```

`Backend/domain` contains business entities. `Backend/application` contains use cases. `Backend/interfaces` and `Backend/repositories` define contracts. `Backend/infrastructure` contains concrete adapters.
